# ROMs

ROM files are not included in this repository. They are copyrighted by their
respective owners and cannot be distributed.

## Expected folder structure

```
ROMS/
├── NINTENDO ENTERTAINMENT SYSTEM GAMES/
│   └── *.nes
├── SUPER NINTENDO ENTERTAINMENT SYSTEM GAMES/
│   └── *.sfc / *.smc
├── NINTENDO GAMEBOY and GAMEBOY COLOR GAMES/
│   └── *.gb / *.gbc
├── SEGA GENESIS GAMES/
│   └── *.md / *.gen
└── SEGA MASTER SYSTEM GAMES/
    └── *.sms
```

Place your legally-owned ROM files in the correct folder and the library
scanner will pick them up automatically on the next server restart.

**Only use ROMs for games you legally own.**
